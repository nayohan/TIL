#### 1번 문제

#### 2번 문제

원소 + 원소 개수 순으로 입력 받으면 이를 올바른 화학식으로 처리하시오.

숫자는 1~10까지 나올 수 있고, 올바르지 않은 경우는 error를 출력한다.

ex) CO12 -> CO2

ex) PSO24 -> P2SO4

ex) NaCl11 -> NaCl
